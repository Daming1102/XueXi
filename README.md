# 《学习》

本项目用于记录从启蒙教学到学术研究的的各科各类的资源内容，包括很多经过实践的优秀学习方法论、各科的人物典故或诸多的历史趣闻等内容。为了培养学习兴趣，更是为了引导志向。本项目会持续更新，直到海枯石烂！
